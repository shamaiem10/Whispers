---

## âœ¨ Features

- **User Authentication**: Secure login and signup system.
- **Whisper Creation**: Share your travel moments with title, content, location, image, and visit date.
- **AI-powered Tags**: Automatic mood, vibe, and emotion tagging.
- **Privacy Control**: Public or private whispers.
- **Explore Section**: Filter public whispers by emotion, mood, or vibe.
- **CRUD Operations**: Create, read, update, delete whispers.
- **Wishlist & ChatLogs**: Save favorite whispers and AI-assisted chat logs.
